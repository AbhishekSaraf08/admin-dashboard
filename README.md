
# Admin Dashboard Web Interface

A responsive **Admin Dashboard** built using **HTML, CSS, SASS, and Bootstrap**.  
This project demonstrates core front-end concepts like responsive layout, forms, tables, media elements, and reusable UI components.

## Features
- Login & Registration Pages  
- Admin Dashboard with cards and canvas chart  
- User Listing Page  
- User Profile Page  
- Settings Page  
- Reusable Sidebar Navigation  
- Fully Responsive Design

## Technologies Used
- HTML5
- CSS3
- SASS
- Bootstrap 5

## How to Run
1. Clone or download the repository  
2. Open `index.html` in your browser
